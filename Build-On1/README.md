## 🌟 Overview

DevPath is a premium, client-side web application built with **React 19** and **Tailwind CSS 4**. It provides an opinionated, structured curriculum for developers learning modern frontend engineering — from beginner HTML to senior-level architecture patterns.

---

## ✨ Features

- **4 progressive levels** — Beginner, Intermediate, Advanced, Senior
- **14 in-depth modules** with multi-part lessons and code examples
- **Interactive quizzes** with instant feedback and deep-dive explanations
- **Level unlock system** — complete a level to unlock the next
- **Persistent progress** — saved automatically to `localStorage`
- **Glassmorphic UI** — premium pastel light theme with custom mesh gradients
- **Back-to-top button** — smooth scroll utility
- **Footer navigation** — click any track to scroll directly to that section
- **Privacy Policy, Terms of Use, and Contact** — modal-based legal pages

## 🚀 Getting Started

### Prerequisites
- Node.js `v18+`
- npm `v9+`

### Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd Build-On1

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173/`.

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.
